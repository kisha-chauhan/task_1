# task_1

🤖 Project 1: Rule-Based AI Chatbot
📖 Overview

The Rule-Based AI Chatbot is a Python-based conversational application that responds to user inputs using predefined rules and conditional statements. Unlike AI-powered chatbots that learn from data, this chatbot follows a set of programmed responses to interact with users.

The chatbot can greet users, remember their names during the conversation, display the current date and time, tell jokes, provide motivational quotes, perform basic arithmetic calculations, count user messages, and save chat history to a text file. This project demonstrates the fundamentals of chatbot development using core Python concepts.



.

🎯 Objectives
Build a simple conversational chatbot using Python.
Understand rule-based chatbot logic.
Practice Python programming fundamentals.
Learn file handling for storing chat history.
Implement user interaction using conditional statements and loops.


✨ Features
👋 Greets the user.
🙋 Stores and remembers the user's name during the session.
⏰ Displays the current time.
📅 Displays today's date.
😂 Generates random programming jokes.
💪 Displays motivational quotes.
😊 Responds to user emotions (Happy/Sad).
🧮 Performs basic arithmetic calculations.
📊 Counts the total number of user messages.
📜 Saves chat history in a text file.
🆘 Displays a help menu containing available commands.
🚪 Safely exits while saving the conversation.

🛠️ Technologies Used
Python 3
datetime module
random module
time module
File Handling

💬 Supported Commands
Command	Description
hi / hello / hey	Greets the user
my name is <name>	Stores the user's name
your name	Displays chatbot name
how are you	Shows chatbot status
time	Displays current time
date	Displays today's date
joke	Shows a random joke
motivate me	Displays a motivational quote
happy	Positive response
sad	Encouraging response
count	Shows total messages
history	Displays saved chat history
help	Displays all available commands
bye	Ends the conversation
